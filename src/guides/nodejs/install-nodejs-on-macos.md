# Installing Node.js on macOS

1. **Install Homebrew** (if you haven't already):
   
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. **Install Node.js**:
   
   brew install node

3. **Verify the installation**:
   
   node -v
   npm -v

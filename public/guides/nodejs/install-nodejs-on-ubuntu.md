# Installing Node.js on Ubuntu

1. **Update the package list**:
   
   sudo apt update

2. **Install Node.js**:
   
   sudo apt install nodejs

3. **Install npm (Node Package Manager)**:
   
   sudo apt install npm

4. **Verify the installation**:
   
   node -v
   npm -v
